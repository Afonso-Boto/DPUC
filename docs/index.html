Our Project

This project aims to develop a new structure of information for the management of syllabuses (Dossier Pedagógico) of UCs.

To achieve this goal, there will be a new process to create and manage UC's syllabuses.
This processes will aid teachers on the creation of UC's syllabuses by providing certain patterns and requirements. This tool is also aimed to aid university workers with approving syllabuses.

Project Plan

  1. Front End
    Development of a Web Apps for University workers, to create and manage syllabuses (DPUC).
  2. Back End
    Reliable, automatized and fast solution to manage DPUCs while incorporating internal services from UA.
  3. Database
    Centralized DPUC Database where all UCs and their syllabuses are stored, other entities such as courses, departments and teachers are also stored here.
  4. Search API
    Fast and effective API that allows for search of UCs.
